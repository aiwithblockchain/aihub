## 一、软件与工具下载

### 1. 浏览器扩展 —— TweetClaw

- **名称**：TweetClaw
- **下载地址**：[Chrome Web Store](https://chromewebstore.google.com/detail/tweetclaw/nolgcgfoebklpejfgebinnakamjamkna)
- **安装方式**：Chrome 浏览器打开链接后，点击「添加至 Chrome」完成安装
- **备注**：每个 Chrome Profile 均需单独安装该扩展

---

### 2. Mac 本地应用 —— OpenHub

- **名称**：OpenHub
- **下载地址**：[Mac App Store](https://apps.apple.com/app/openhub/id6761117317)
- **安装方式**：点击链接后在 Mac App Store 中下载安装
- **备注**：每台 Mac 电脑均需安装

## 四、Chrome 浏览器配置

### 5. Chrome Profile 与扩展配置

每台 Mac 电脑按以下步骤配置：

- [ ] 安装 Google Chrome 浏览器
- [ ] 创建 **5 个** Chrome Profile（配置文件）
  - 创建方式：Chrome 右上角头像 → 「添加」→ 新建配置文件
- [ ] 每个 Profile 分别安装 **TweetClaw** 扩展
  - 安装地址：[Chrome Web Store - TweetClaw](https://chromewebstore.google.com/detail/tweetclaw/nolgcgfoebklpejfgebinnakamjamkna)
- [ ] 每个 Profile 登录对应的 Twitter/X 账号

**汇总：**

| 设备 | Profile 数量 | TweetClaw 安装数量 |
|------|-----------|--------------------|
| Mac #1 | 5 | 5 |
| Mac #2 | 5 | 5 |
| **合计** | **10** | **10** |

## 五、开发环境（可选）

### 6. VS Code 与 Python 环境

每台 Mac 电脑按需安装：

- [ ] **VS Code**（可选）
